Política de Privacidad

Esta aplicación permite a administradores de páginas de Facebook eliminar sus propias publicaciones usando la API oficial de Meta.

Datos que usamos

Solo accedemos a las publicaciones de las páginas que el usuario administra. No almacenamos, vendemos ni compartimos ningún dato personal.
